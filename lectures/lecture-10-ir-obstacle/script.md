# 第 10 课：红外循迹小车

## 一、红外传感器原理

### ① 从 "手电筒照黑线" 引入

红外循迹模块的工作原理其实很简单。想象一下：你在黑暗的房间里拿手电筒照地面——

- 照到**白色地面**，光线强烈反射回来，你能看清
- 照到**黑色胶带**，光线被吸收，一片漆黑

红外循迹模块就是利用这个原理：

| 地面颜色 | 反射强度 | 模块输出 | 含义 |
| :--- | :--- | :--- | :--- |
| **白色** | 强 | **低电平（0）** | 检测到白地 |
| **黑色** | 弱 | **高电平（1）** | 检测到黑线 |

> [!WARNING]
> **不同模块输出逻辑可能相反！** QX_A11 小车上的模块是：**白=0，黑=1**。接线前务必先验证。

### ② 模块结构拆解

常见的 TCRT5000 红外模块有 4 个引脚：

| 引脚 | 功能 | 说明 |
| :--- | :--- | :--- |
| VCC | 电源 | 3.3V-5V |
| GND | 地 | 共地 |
| DO | 数字输出 | 比较器输出，0 或 1 |
| AO | 模拟输出 | 反射强度电压（本实验用不到） |

核心是 **LM393 比较器**：
- 红外发射管发光 → 接收管收到反射光 → 产生电压 → 与电位器设定的阈值比较 → 输出高/低

> [!TIP]
> **电位器的作用**：模块上那个蓝色小旋钮用来调节检测距离。顺时针调灵敏度降低（检测更远），逆时针调灵敏度提高（只能检测很近）。

### ③ 小车上的循迹传感器

QX_A11 小车上用于循迹的模块有 **2 个**：

| 传感器 | 引脚 | 说明 |
| :--- | :--- | :--- |
| 左循迹 (left_led1) | P3^5 | 检测左侧黑线 |
| 右循迹 (right_led1) | P3^4 | 检测右侧黑线 |

> [!NOTE]
> `left_led1` / `right_led1` 是历史遗留命名，实际功能是红外传感器，不是 LED。

## 二、循迹逻辑

### 核心思路：4 种状态

把左右传感器的读数拼成一个数，根据这个数决定动作：

```c
// 把两个传感器的状态拼成一个两位数
char data1, data2, data3;
data2 = left_led1;      // 左：0=白，1=黑
data3 = right_led1;     // 右：0=白，1=黑
data1 = data2 * 10 + data3;  // 拼成：10/11/01/00
```

| data1 值 | 左传感器 | 右传感器 | 小车动作 | 含义 |
| :--- | :--- | :--- | :--- | :--- |
| **11** | 黑 | 黑 | 前进 | 在线中间，正常走 |
| **10** | 白 | 黑 | **右转** | 偏左了，往右纠 |
| **01** | 黑 | 白 | **左转** | 偏右了，往左纠 |
| **00** | 白 | 白 | **停车** | 离线了，停下来 |

> [!TIP]
> **为什么差速转向？** 左右轮速度不相等，产生差速让小车转弯。速度差越大，转弯越急。

## 三、代码实现

### 第 1 步：引脚定义

引脚定义在 [config.h](code/config.h) 中：

```c
// config.h
sbit left_led1   = P3^5;   // 左循迹传感器（白=0，黑=1）
sbit right_led1  = P3^4;   // 右循迹传感器（白=0，黑=1）
```

> [!TIP]
> 循迹传感器接在 P3 口，不占用 P1（电机 PWM）和 P2（超声波），互不冲突。

### 第 2 步：读传感器

```c
// 读传感器状态
char data2 = left_led1;    // 读左传感器
char data3 = right_led1;   // 读右传感器
```

### 第 3 步：判断 + 执行

```c
// 拼接状态
char data1 = data2 * 10 + data3;

// 根据状态决定动作
if (data1 == 11) {
    forward(120, 120);     // 两线在线 → 前进
}
else if (data1 == 10) {
    left_run(80, 160);     // 左白右黑 → 右转纠偏
}
else if (data1 == 1) {
    right_run(160, 80);    // 左黑右白 → 左转纠偏
}
else {
    stop();                // 两线都离线 → 停车
}
```

## 四、实操注意事项

1. **调电位器**：每个红外模块上的电位器先调至中间位置，再根据实际路面微调
2. **环境光影响**：强光（阳光、聚光灯）会干扰红外传感器，室内灯光影响较小
3. **黑线宽度**：建议 1.5-2cm，两条线间距 3-5cm
4. **安装高度**：红外模块离地面高度建议 1-2cm，太高灵敏度下降


烧录后，将小车放在黑线上，观察是否按预期循迹。

### 任务二：调试速度参数（必做）

观察循迹效果，调整 `left_run()` 和 `right_run()` 中的速度参数：

- **转弯太急**：减小左右轮速度差
- **转弯太缓**：增大左右轮速度差
- **抖动严重**：让转向更平滑（速度差越小越平滑，但转向越慢）

### 任务三：速度自适应（选做）

根据偏离程度调节速度差：

| 偏离程度 | 左右轮速度差 | 说明 |
| :--- | :--- | :--- |
| 轻微偏离 | 差值 40 | 小幅度纠偏 |
| 明显偏离 | 差值 80 | 大幅度纠偏 |
| 完全离线 | 停车 | 重新寻找黑线 |

### 任务四：原地旋转搜索（选做）

当两传感器都离线（data1 == 0）时，不要直接停车，改为原地旋转搜索黑线：

```c
else {  // 两线都离线
    left_run(100, 100);  // 原地旋转
    Delay_Ms(200);
}
```
