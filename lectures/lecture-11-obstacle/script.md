# 第 11 课：红外避障小车

## 一、红外避障传感器

### ① 和循迹传感器的区别

红外避障模块和循迹模块**硬件相同**（都是 TCRT5000），但**使用场景不同**：

| | 循迹模块 | 避障模块 |
| :--- | :--- | :--- |
| **检测目标** | 地面黑线 | 侧方障碍物 |
| **安装方向** | 朝下 | 朝侧方 |
| **检测距离** | 1-2cm | 2-30cm |
| **电位器调节** | 灵敏度调高 | 灵敏度调低（检测更远） |

> [!NOTE]
> 电位器调节方向相同：顺时针调灵敏度降低（检测更远），逆时针调灵敏度提高（检测更近）。

### ② 小车上的避障传感器

QX_A11 小车上用于避障的模块有 **2 个**：

| 传感器 | 引脚 | 说明 |
| :--- | :--- | :--- |
| 左避障 (left_led2) | P3^6 | 检测左侧障碍物 |
| 右避障 (right_led2) | P3^7 | 检测右侧障碍物 |

> [!NOTE]
> `left_led2` / `right_led2` 是历史遗留命名，实际功能是红外避障传感器。`led1` 系列用于循迹，`led2` 系列用于避障。

### ③ 输出逻辑

避障模块的输出逻辑与循迹模块**可能不同**，QX_A11 小车上：

| 状态 | 输出 |
| :--- | :--- |
| **无障碍** | **高电平（1）** |
| **有障** | **低电平（0）** |

> [!WARNING]
> **务必先验证你的模块输出逻辑！** 有些模块是反过来的：有障=1，无障碍=0。用万用表或模块上的 LED 测试。

## 二、避障逻辑

### 核心思路：3 种状态

```c
// 读避障传感器
char left = left_led2;     // 左：0=有障，1=无障碍
char right = right_led2;   // 右：0=有障，1=无障碍

// 根据状态决定动作
if (right == 0) {
    // 右侧有障 -> 向左转
    left_run(120, 120);
}
else if (left == 0) {
    // 左侧有障 -> 向右转
    right_run(120, 120);
}
else {
    // 两侧都无障碍 -> 前进
    forward(120, 120);
}
```

| 左侧 | 右侧 | 小车动作 | 含义 |
| :---: | :---: | :--- | :--- |
| 1 (无障碍) | 0 (有障) | **左转** | 右侧碰到墙，向左避开 |
| 0 (有障) | 1 (无障碍) | **右转** | 左侧碰到墙，向右避开 |
| 1 (无障碍) | 1 (无障碍) | **前进** | 前方无障碍 |
| 0 (有障) | 0 (有障) | **后退** | 两侧都有障，后退找路 |

### 红外避障 vs 超声波避障

| | 红外避障 | 超声波避障 |
| :--- | :--- | :--- |
| **检测距离** | 2-30cm（近） | 2-400cm（远） |
| **反应速度** | 快（直接读电平） | 慢（需要发射+等待回波） |
| **适用场景** | 贴墙走、精细避障 | 远距离预警 |
| **优点** | 反应快、成本低 | 距离远 |
| **缺点** | 距离近、受颜色影响 | 对软/斜面物体无效 |

> [!TIP]
> **实际小车通常两者配合使用**：超声波远距离预警，红外近距离精细避障。

```c
// config.h
```

### 第 2 步：读传感器

```c
// 读避障传感器
char left = left_led2;     // 0=有障，1=无障碍
char right = right_led2;   // 0=有障，1=无障碍
```

### 第 3 步：判断 + 执行

```c
if (right == 0) {           // 右侧检测到障碍物

else if (left == 0) {       // 左侧检测到障碍物
    right_run(120, 120);   // 向右转避开
}
else {                      // 两侧都无障碍
    forward(120, 120);     // 前进
}
```

## 四、实操注意事项

1. **验证输出逻辑**：用手靠近传感器，看模块上 LED 是否亮/灭，确认输出是 0 还是 1
2. **调电位器**：避障检测距离建议调到 5-15cm，太远会提前转向，太近会撞上
3. **侧面安装角度**：避障模块应朝侧前方 30-45 度安装，不要正对侧面
4. **障碍物颜色**：黑色/吸光物体反射弱，检测距离会缩短

### 任务二：处理双侧同时有障（必做）

当左右两侧同时检测到障碍物时，增加后退+转向逻辑：

```c
if (right == 0 && left == 0) {
    // 两侧都有障 -> 后退 + 转向
    back_run(120, 120);
    Delay_Ms(300);
    left_run(120, 120);
    Delay_Ms(200);

```

### 任务三：红外+超声波混合避障（选做）

结合第 8-9 课的超声波模块，实现远近双重避障：

- **红外**（2-30cm）：侧方障碍物快速反应
- **超声波**（30-300cm）：前方障碍物提前预警

```c
// 伪代码
if (红外检测到侧方障碍) {
    快速转向();            // 贴墙绕行
}
else if (超声波检测到前方障碍) {
    后退 + 转向();         // 脱困
}
else {
    forward();             // 正常前进
}
```
